# 🤖 LearnMate – AI for Personalized Course Pathways

## 📌 Overview
LearnMate is an AI-powered assistant designed to provide personalized learning pathways for students. It analyzes user interests and skill levels to recommend tailored courses and career guidance.

## 🚀 Problem Statement
Students often struggle to identify the right learning path due to lack of personalized guidance. This project addresses that gap by providing AI-driven recommendations based on individual goals and preferences.

## 💡 Solution
Developed an intelligent AI assistant that:
- Understands user inputs (interests, skill level)
- Generates customized course pathways
- Guides students toward structured career growth

## 🛠️ Features
- Personalized course recommendations  
- Interactive AI-based guidance system  
- Real-time response generation  
- User-friendly interface  

## ⚙️ Technologies Used
- IBM Watsonx.ai  
- Granite Model  
- IBM Cloud  
- Prompt Engineering
